# Card Design

Esse é um projeto simples que utiliza html, css e algumas funções do javascript para criar efeito parallax em uma imagem.

Para este efeito funcionar, devemos ter varias partes da imagem separada. Com isso juntamos todas elas para montar a imagem em sí.

E com **JavaScript** conseguimos mover cada parte individualmente conforme vamos descendo a tela para baixo. Utilizando o parametro **scroll**.
